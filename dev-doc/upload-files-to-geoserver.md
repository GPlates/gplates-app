### Upload a raster file

#### Method 1: use GeoServer web page

- make sure the .tif file is properly georeferenced
- upload the .tif file to the server via SFTP
- create a "coveragestore" in the web page
- create and publish a layer from the "coveragestore" in the web page
- check the "layer preview"
